# Client Images Organization Guide

Please move images to the appropriate folders based on what they show:

## Folder Structure

- **Authorization/** - Login, signup, authentication screens
- **Home/** - Home screen, main dashboard views
- **CategoryButtons/** - Category selection, navigation buttons

## How to Organize

1. **Option 1: Manual Organization**
   - Move images to the appropriate subfolder
