## Image Specifications

- **Size**: Phone-sized screenshots (recommended width: 300px)
- **Format**: PNG or JPG
- **Aspect Ratio**: Mobile phone aspect ratio (approximately 9:16 or similar)
- **Style**: Screenshots should show the actual app interface

## Tips

- Use actual phone screenshots from your app
- Ensure images are clear and readable
- Keep file sizes reasonable for web viewing
- Images will automatically wrap to multiple rows on smaller screens
